# Frontend Applications

This folder contains all frontend applications for AI Budgeting Coach.

## Applications

### web/
React 18 + TypeScript web application

**Technology:**
- React 18.3+
- TypeScript 5.3+
- Material-UI (MUI) 5.15+
- Zustand (state management)
- TanStack Query (data fetching)
- React Router 6+
- Vite (build tool)

**Features:**
- User authentication (login/register)
- Transaction management (CRUD, CSV import)
- Budget tracking with visual progress
- AI-powered insights display
- Notifications center
- Dashboard overview

See `web/README.md` for detailed setup and development instructions.
